## 출처 1: 사용자가 채팅으로 제공한 L-Acoustics 공식 홈페이지 텍스트

- Fixed Install

**Electroacoustic specs**

- 2-way passive enclosure
- amplified by LA7.16i / LA2Xi / LA4X / LA12X
- Usable bandwidth (-10 dB):
  - 60 Hz – 20 kHz ([SOKA_60])
  - 100 Hz – 20 kHz ([SOKA])
  - 200 Hz – 20 kHz ([SOKA_200])
- Maximum SPL¹:
  - 124 dB ([SOKA_60])
  - 130 dB ([SOKA])
  - 133 dB ([SOKA_200])
- Nominal directivity (-6 dB): vertical +5°/-21° / horizontal 140°
- Transducers:
  - MF: 9 × 3.5″ cone drivers
  - HF: 3 × 1″ diaphragm compression drivers
- Acoustical load: LF closed enclosure / HF DOSC waveguide, L-Fins
- Nominal impedance: 8 Ω

¹ Peak level measured at 1 m under free field conditions using pink noise with crest factor 4 (preset specified in brackets).

**Physical specs**

- Connectors: 1 x 4-point screw terminal
- Rigging and handling: Soka inWall includes all elements for integration in plain walls
- Dimensions: 99 mm (3.9 in) / 1065 mm (41.9 in) / 99 mm (3.9 in)
- Weight (net): Soka inWall: 11,7 kg / 25,8 lb
- Cabinet: premium grade Baltic beech and birch plywood
- Front: Coated steel grill / Acoustically neutral 3D fabric
- Finish: dark grey brown Pantone 426 C / pure white RAL 9010 / custom RAL code on special order
- IP: IP55
